Over the Moon!

Over the Moon! is an app designed to allow climbers to catalog and share bouldering problems on system walls, which are walls with a high density of holds, and no predetermined routes. The app will eventually allow users to upload pictures of their wall, and create the route within the app.

This project is not yet done

I started by designing this as a website, this was one of the first sketches I made for the project.Eventually, I began working on the site, shortly, though, I decided that Over The Moon should be developed as a mobile app, rather than a website.

Which has led me to making these sketches, which will soon be redone with wireframing software, and animated with Invision.

After that, I'll develop the app for Android and IOS using React Native, but that will have to wait until I have more free time.

The Biggest Challenge

Really, the only challenge I've faced with this project is that I've never worked with React Native. To fix this, I started learning it with online tutorials plus trail and error.
